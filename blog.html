How to AngularJS

Notes:

This tutorial assumes you have a working knowledge of HTML, CSS, JS, JQuery, Json, Bootstrap, and AngularJS.

Part 1 -- Using Angular to extract Json data

Step 1

Set up your ng-app.

For example:

var app = angular.module('listJson', []);

Step 2

Set up your controller to use the $scope and $http services.

For example:

app.controller('ListController', ['$scope','$http', function($scope,$http)

Step 3

Initialize $scope = this and initialize an array as part of $scope to hold your Json.

For example:

var $scope = this;
$scope.items = [];

Step 4

Use the $http get service to extract your Json and, on success, assign the data to your array.

For example:

$http.get('tutorials.json').success(function(data){
$scope.items = data;

Step 5

Now that your angularJS is ready, add it to your html page with the ng attributes. Be sure to include ng-app and ng-controller. To iterate through your Json objects, use ng-repeat.

Ending notes

Be sure to include all necessary script and link references. Be sure to use ng-src for images and ng-href for links obtained through Angular. Style to desire.
